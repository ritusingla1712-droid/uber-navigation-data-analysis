Q1 · Basic Level
Distinct Values
What are & how many unique pickup locations are there?
SELECT DISTINCT pickup_location FROM ride1;
ReturnsAll unique locations
💡DISTINCT eliminates duplicate rows. Wrap with COUNT() to get the total number of unique values.

Q2 · Basic Level
Aggregate Count
What is the total number of rides in the dataset?
SELECT COUNT(*) FROM ride1;
ReturnsTotal row count
💡COUNT(*) counts all rows including NULLs. Use COUNT(column) to skip null values.

Average Calculation
Calculate the average ride duration.
SELECT AVG(ride_duration) FROM ride1;
ReturnsMean ride duration (mins)
💡Use ROUND(AVG(col), 2) for cleaner output. AVG ignores NULL values automatically.

Q4 · Basic Level
Top-N Query
List the top 5 drivers based on their total earnings.
SELECT driver_id, driver_name, earnings FROM driver1 ORDER BY earnings DESC LIMIT 5;
ReturnsTop 5 highest earners
💡ORDER BY ... DESC + LIMIT is the standard Top-N pattern. Always sort before limiting!

Q5 · Basic Level
Group By
Total number of rides for each payment method.
SELECT payment_method, COUNT(*) AS total_rides FROM ride1 GROUP BY payment_method ORDER BY total_rides DESC;
ReturnsRides per payment type
💡GROUP BY collapses same-value rows into one. Combine with COUNT, SUM, AVG for summaries.

SELECT ride_id, fare_amount FROM ride1 WHERE fare_amount > 20;
ReturnsHigh-fare rides only

SELECT pickup_location, COUNT(*) AS rides FROM ride1 GROUP BY pickup_location ORDER BY rides DESC LIMIT 1;

Combined Aggregates
Count rides & avg distance from 'Dhanbad' pickup.

ReturnsCount + avg for Dhanbad

Q14 · Basic Level
JOIN + GROUP BY
List passengers who have taken the most number of rides.
SELECT p.passenger_name, COUNT(r.ride_id) AS total_rides FROM passenger1 p JOIN ride1 r ON p.passenger_id = r.passenger_id GROUP BY p.passenger_id, p.passenger_name ORDER BY total_rides DESC;
ReturnsRiders ranked by usage
💡JOIN links tables on a shared key. Include all non-aggregated columns in GROUP BY!

Q19 · Basic Level
Percentage Distribution
Calculate % distribution of rides per pickup location.
SELECT pickup_location, COUNT(*) AS rides, ROUND(100.0 * COUNT(*) / (SELECT COUNT(*) FROM ride1), 2) AS pct FROM ride1 GROUP BY pickup_location ORDER BY pct DESC;
ReturnsLocation share %
💡Use 100.0 (not 100) to avoid integer division truncation. Scalar subquery runs once for total.

Key Takeaways
🔍
SELECT + DISTINCT
Filter unique values and eliminate duplicates from result sets.
📊
COUNT / SUM / AVG
Core aggregates for counting, summing and averaging across rows.
🧩
GROUP BY + ORDER BY
Group data into buckets and sort results for ranked insights.
🔗
JOIN
Combine rows from two tables using a shared key column.
🎯
WHERE + HAVING
WHERE filters rows; HAVING filters grouped/aggregated results.
⚡
Subqueries
Nest a SELECT inside another to compute dynamic thresholds.
